## 📌 프로젝트 개요
### 프로젝트 명
덕후감(Deokhugam)

### 프로젝트 소개
덕후감은 책 읽는 즐거움을 공유하고, 지식과 감상을 나누는 책 덕후들의 커뮤니티 서비스입니다.  
도서 이미지 OCR 및 ISBN 매칭 서비스를 제공합니다.

### 프로젝트 정보
- **기간**: 2026.02.27 ~ 2026.03.23
- **인원**: 5명
- **역할**: 도서 커뮤니티 서비스의 Backend 개발
